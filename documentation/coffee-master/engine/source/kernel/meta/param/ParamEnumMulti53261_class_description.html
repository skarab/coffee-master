Description
Custom edition of uint32 binded on enum.
Tell the GUI system the specified uint32 is binded to an enum MULTI, read boolean array. Used to overpass the enum system, since an enum can be SINGLE or MULTI, this is to allow uint32 to use a MULTI approach on an enum in SINGLE mode.
