coffee::resource::Manager ( Class )
Resource manager module.
 
Description
Resource manager module.
This is the entry point of the resource system. Usage is to call Object = Load to instantiate the resource, IsAvailable() will tell if the resource is ready, it can even check for a specific LOD.
