Description
Resource handler.
Its an helper that you may attach to a resource::Object, to get notifications about loading states (ex: if you wish to trigger something when your resource is reloaded).
Notes
  • It doesnt store a reference, not its serializable, its an helper you attach to an already existing resource::Object.
