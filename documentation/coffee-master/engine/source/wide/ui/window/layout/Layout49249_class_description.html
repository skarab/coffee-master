coffee::ui::Layout ( Class )
Window Layout class.
 
Description
Window Layout class.
Each window own his own Layout instance. Disabled by default, it can automatically sort and move/resize window or children depending of the flags set. Performing various operation as Shrink to content, Expand, sort and stick children ...
