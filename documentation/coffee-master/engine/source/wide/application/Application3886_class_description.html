coffee::Application ( Class )
Heart of each Coffee application.
 
Description
Heart of each Coffee application.
This is an interface to inherit in each new project. It contains the project configuration and the application controls, and gives access to useful callbacks... Basically it's only a simple single-threaded singleton module, handled automatically by the module manager. It has shell::MODULE_ATTRIBUTE_Defaults properties.
